Automatic payment setup

This article will guide you on how to Set up Coinpayment for automatic deposits and withdrawal in Online trade.
What is Coinpayment

CoinPayments is a payment processor that allows merchants to accept Bitcoin and hundreds of altcoins in their online store or web portal  through easy to use plugins, APIs and POS interfaces. With over 2.2 million users across 182 countries around the world, CoinPayments.net is most comprehensive multi-cryptocurrency platform available online.

Coinpayment setups:
  1. Create a Coinpayment account Here if you dont have
  2. Goto Account under API keys
  3. Create APP KEY and API key
  4. Click on Edit Permission
  5. Under API keys Permissions
  6. Check to this three commands: create_withdrawal, txn_info, create_transaction.
  7. After that, click on Update Permission to save Changes
  8. Lastly, supply the required credentials: Public Key, Private key and Merchant ID to Online Trade coinpayments settings. You are done!
